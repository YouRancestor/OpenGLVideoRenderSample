# OpenGLVideoRenderSample
OpenGL渲染视频样例，依赖VLC、Qt。

## Build
```shell
qmake VlcPlayerWidget.pro INCLUDE+=path/to/libvlc/include LIBS+=path/to/libvlc/lib
make # or mingw32-make for mingw or jom for msvc ...
```
